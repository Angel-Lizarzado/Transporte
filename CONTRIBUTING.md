# Guía de Contribución

Este es un proyecto privado. Para contribuir:

1. Contactar al administrador del repositorio
2. Obtener acceso al repositorio
3. Crear una rama para tu contribución
4. Seguir las convenciones de código existentes
5. Asegurarse de que todos los tests pasen
6. Crear un pull request con una descripción clara

## Convenciones de Código

- Usar TypeScript para todo el código
- Seguir las convenciones de Next.js 14
- Usar Tailwind CSS para estilos
- Mantener componentes pequeños y reutilizables
- Documentar funciones complejas
